# axiom-battle-api

WuDao Axiom Battle REST API

## Status

## Run locally

```bash
pip install -r requirements.txt
python main.py
```

## Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/` | GET | Service info |
| `/battle` | POST | Axiom battle |
| `/arena` | GET | Arena status |
| `/health` | GET | Health check |
